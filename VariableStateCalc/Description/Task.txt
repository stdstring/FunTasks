На вход Вашей программе в качестве параметра командной строки подается путь к файлу,
содержащий код программы на языке C#. Примеры возможных входных данных в приложении к письму Sample1.cs и Sample2.cs.

Для простоты можно считать, что файл всегда содержит только определение единственного класса Program, который содержит определение двух методов Main и Evaluate.

Метод Main содержит вызовы метода Evaluate c произвольными значениями аргументов и выводит их на экран. В остальном, содержание Main нас не интересует.

Метод Evaluate принимает произвольное количество булевых аргументов. Количество аргументов от 0 до 50.
Первая строка метода Evaluate всегда содержит объявление целочисленной переменной x, последняя - возврат значения. Остальное тело метода состоит из операторов if и присвоений переменной X целочисленных значений.

Задание 1
Нужно вывести на консоль список возможных состояний переменной в конце метода Evaluate, которые могут быть выведены на экран в Main.

Для примера из Sample1.cs:
OUTPUT:
[1, 4, 5, 6].
